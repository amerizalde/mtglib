## Rules Text
Whenever a creature you control with a +1/+1 counter on it leaves the battlefield, create a Mutagen token for each +1/+1 counter on it. (A Mutagen token is an artifact with "1, Tap, Sacrifice this token: Put a +1/+1 counter on target creature. Activate only as a sorcery.")
Tap: Exile target card from a graveyard. Create a Mutagen token.
